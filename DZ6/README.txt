## Домашнее задние №6

Реализован поиск по данным "Город — население".

Функциональность:
- поиск по названию города;
- поиск по населению;
При поиске по населению установлена защита от отрицательного числа.

Используются:
- Razor Pages;
- методы OnGetByCity и OnGetByPopulation;
- директива маршрута @page "{handler?}".

P.S. @page "{handler?}" — это директива в ASP.NET Core Razor Pages, определяющая маршрут (URL) страницы и разрешающая необязательный параметр {handler} в строке запроса. 
Это позволяет вызывать конкретные методы-обработчики (например, OnGetSite, OnPostDelete) внутри PageModel.
