# DZ3 — Города (ASP.NET Core + Autofac)

Учебное задание по ASP.NET Core.

Реализовано:
- маршруты для городов (координаты, информация, памятники)
- Dependency Injection через Autofac
- хранение данных в текстовых файлах
- отображение изображений из wwwroot

Запуск: открыть `DZ3.sln` и запустить проект.
