# DZ2 — Вычисление sin(x) с использованием Dependency Injection

Учебный проект ASP.NET Core (.NET 8).

## Описание
В проекте реализовано вычисление функции sin(x):
- с использованием встроенных методов класса Math
- с использованием ряда Тейлора



## Используемые технологии
- ASP.NET Core
- .NET 8
- Autofac
- Dependency Injection
